Quick start

Get the overlay on screen in a few steps.

1

Download & run

Download Keystrokes, extract, and run the executable. No installer required.

2

Add to OBS

In OBS, add a new Window Capture source and select the Keystrokes window. Apply a chroma key filter to the source to remove the window background so only the keys are visible.

3

Position & stream

Move the overlay where you want it, then start your stream. Keys and mouse show in real time.
